# Ecloire — static storefront

Fine jewelry marketing site: home, shop (placeholder catalog), product detail, about, contact, FAQ, shipping & returns, privacy, and terms.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx --yes serve .
```

## Brand

See `BRAND.md` and `images/README.md`.

## Structure

- `css/` — global styles and logo lockup.
- `js/` — product data (`EcloireProducts`), shop list, home featured, shared behavior.
- `images/` — SVG mark and lockup.
